# sfc gamepad decoder
## overview
スーパーファミコン用のゲームパッドを読み取ってパラレルの信号に変換します。
ゲームパッドの検出する周期は約1/60秒（16ミリ秒）に設定します。このため、最大で1/60秒のタイミングのずれが生じる可能性があります。気になる場合はプログラムを修正して、周期を短くしてみてください。
## Files
sfcpaddecoder.ino : ソースコード。Arduino IDE用。
